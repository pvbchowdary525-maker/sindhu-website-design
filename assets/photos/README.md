# Photos

Drop your own top-down biryani photos here. The site picks them up by filename:

| Filename | Used for |
|---|---|
| `dum-biryani.jpg` | Menu card 1 |
| `fry-biryani.jpg` | Menu card 2 |
| `mix-biryani.jpg` | Menu card 3 |
| `story.jpg` | Editorial frame in the "Mana Katha" section |

If a file is missing, the menu card falls back to the SVG placeholder
(`assets/biryani-placeholder.svg`) so the layout never breaks.

## Shooting tips

- Top-down, phone held flat over the plate.
- Banana leaf, brass plate, or a textured wood surface — anything but white tile.
- Window light, late afternoon. No flash.
- Square or 4:5 portrait crop (the card mask is roughly square via the
  organic-blob border-radius).
- Slight steam in frame is great. Slight motion blur on raita drip is fine.
